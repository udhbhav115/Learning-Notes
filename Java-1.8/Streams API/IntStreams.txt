-> What is IntStream.?
IntStream is a sequence of primitive int values which is an interface that extends the BaseStream interface.

-> What are static methods present in IntStream interface.?
Range(int startInclusive,int endExclusive)
RangeClosed(int startInclusive,int endInclusive)
allMatch(IntPredicate predicate)
noneMatch(IntPredicate predicate)
anyMatch(IntPredicate predicate)
iterate(int startIndex,incrementCounter).limit(no_of_values)
builder() it will convert primitive to object
generate(Supplier interface) IntStream.generate(()->100).limit(5) it will insert 100 5 times into the IntStream

----------------------------------------------pseudocodes------------------------------------------------

OptionalInt max = IntStream.of(10,20,30,40,50).filter(i->i<0).max();
System.out.println(max); // output : OptionalInt.empty

int max = IntStream.of(10,20,30,40,50).filter(i->i<0).max().getAsInt();
System.out.println(max); // In this case it will give noSuchElementException because there are no negative numbers in the stream
